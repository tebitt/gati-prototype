# Frontend

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

# Backend

## Project setup
```
pip install -r backend/requirements.txt
```

### Compiles and hot-reloads for development
```
uvicorn backend.server:app --reload
```
